# xsmd.nvim

This is a plugin that provides an easy way to use [xsmd](https://github.com/RaquerLabs/xsmd/) in neovim.

## Installation

You can use nvim's package manager to install this plugin.

```lua
vim.pack.add({
  { src = "https://github.com/RaquerLabs/xsmd.nvim" },
})
require("xsmd").setup()
```
